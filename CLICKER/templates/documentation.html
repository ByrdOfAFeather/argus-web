1Keyboard Shortcuts

ARGUS uses a variety of keyboard shortcuts to allow for speedy and accurate work.

When hovering over a video (notice a thin black border) with the page focused:

x - Zoom out

z - Zoom in

L - Lock to epipolar lines (This will let you only click points on the present epipolar lines)

SPACE - clicks a point with the location noted in the zoom window

ENTER - clicks a point with the location noted in the zoom window

Arrow Keys - Allows for precise movement of the zoom window (allows for precise clicking)

ALT + Arrow Keys - Moves the zoom window by a very small amount

SHIFT + Arrow Keys - Moves the zoom window by a large amount

5Exporting Points

Exporting points can export a variety of point types depending on the settings. Since multiple .csv files could be created when exporting points, all files are placed into a .zip file before the download.

For 1-camera projects:

If you have the origin set (via the set-origin button), all points will be exported in their raw coordinates as well as their origin-based coordinates with proper units from the origin. If the origin is not set, only the raw coordinates are set. The raw coordinates have origin in the upper left corner with down, right being the positive direction. The exported points will have rows representing the frames of the videos and the columns being the x,y coordinates.

For 2-camera projects:

If you have DLT Coefficients loaded, 2D and 3D points will be exported. 3D points will only have non-null values for frames where every video has a point. The exported points will follow the same format mentioned above.
